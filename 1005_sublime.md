## 使用相关
- Sublime Text的默认设置是不开启显示编码的，如果想开启，可通过菜单Perference → Settings

  在打开的配置文件里 ，在大括号后面，增加以下内容：
  
  `// Display file encoding in the status bar`
  
  `"show_encoding": true,`
  
  `// Display line endings in the status bar`
  
  `"show_line_endings": true,`
  
  保存配置文件。
